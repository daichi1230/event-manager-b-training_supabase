# Event Manager B+ Supabase コピペ手順

## 1. このフォルダをそのまま配置
```powershell
cd C:\work
mkdir event-manager-b-supabase
```

この ZIP を展開して、そのフォルダに入ってください。

## 2. Supabase で DB を作る
1. Supabase プロジェクトを作成
2. `supabase/schema.sql` の中身を SQL Editor に丸ごと貼る
3. 実行する

## 3. `.env.local` を作る
PowerShell:

```powershell
Copy-Item .env.example .env.local
notepad .env.local
```

以下を入れて保存:

```env
VITE_SUPABASE_URL=https://YOUR_PROJECT_REF.supabase.co
VITE_SUPABASE_PUBLISHABLE_KEY=YOUR_SUPABASE_PUBLISHABLE_KEY
```

## 4. 起動
```powershell
npm install
npm run dev
```

## 5. ブラウザで開く
```text
http://localhost:5173
```

## 6. 最初の管理者を昇格
1. 画面から `your-admin@example.com` で新規登録
2. Supabase の SQL Editor で次を実行

```sql
select public.promote_user_to_admin('your-admin@example.com');
```

3. ログアウト → 再ログイン
4. 管理者メニューが見えたら成功

## 7. 本番ビルド確認
```powershell
npm run build
npm run preview
```

## 8. よくあるつまずき
- `.env.local` を作っていない
- Publishable key ではなく service_role key を入れてしまった
- メール確認が有効で、サインアップ後に未確認のままログインしようとしている
- 管理者昇格 SQL をまだ流していない
